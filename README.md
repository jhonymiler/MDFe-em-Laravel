## Sobre o Sistema MDF-e

Se trata de um simples sistema para emissão de MDF-e onde é possível:

- Encerrar MDF-e emitidos.
- Enviar novos MDF-e.
- Cancelar MDF-e.

Feito com o Framework Laravel com fácil manutenção.
